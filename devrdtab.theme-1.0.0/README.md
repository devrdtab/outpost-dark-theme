<h1 align="center">Outpost Dark Theme</h1>

A dark Visual Studio Code theme with muted contrast, warm rust accents, and a quiet editor interface.

## Preview

![Outpost Dark Theme preview](images/preview.png)

![Outpost Dark Theme secondary preview](images/preview2.png)

## Installation

1. Download `outpost-1.0.0.vsix` from the latest GitHub Release.
2. Open Visual Studio Code.
3. Go to Extensions.
4. Open the `...` menu.
5. Choose `Install from VSIX...`.
6. Select the downloaded `.vsix` file.
7. Select `Outpost Dark` from Color Theme settings.

## Manual Install From Terminal

```bash
code --install-extension outpost-1.0.0.vsix
```

## What's Included

- Dark workbench colors
- Muted editor contrast
- Warm accent colors
- Syntax highlighting for common language scopes
- Terminal color palette
